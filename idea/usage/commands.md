# Commands

| Command | Description | Permission |
|---------|-------------|------------|
| `/discord` | Sends the configured Discord invite in chat and, for players, can also show an action bar and play the pling sound. | `ezdonutdiscord.discord` |
| `/discord reload` | Reloads `config.yml` and refreshes PlaceholderAPI integration state. | `ezdonutdiscord.reload` |

> Tip: The `reload` tab completion is only shown to senders who already have reload access.
